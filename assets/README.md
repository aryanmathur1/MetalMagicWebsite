Asset folders for the Metal Magic website.

Drop photos into the matching folder using the filenames already referenced in the HTML files. The site will show styled placeholders until the real files are added.

- brand: team logo and brand files
- team: meet-the-team headshots
- outreach: outreach event photos
- hackathons: extra gallery photos for Code Crush events
- robot: robot and mechanism photos
- awards: award and competition photos
