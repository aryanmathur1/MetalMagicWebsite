# Outreach Page Photos

Photos go in `assets/outreach/`. Use **landscape orientation** when possible.  
Each event card has **3 slots**: one large primary photo (full width) and two smaller ones below.  
If any photo file is missing the site automatically shows a styled purple placeholder — no broken images.
